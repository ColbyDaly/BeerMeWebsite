Keyword
ABV

Alcohol By Volume ("+10" will return everything above 10%, "-10" will return everything under 10%, "8,10" will return everything between 8% and 10% inclusive)

IBU

International Bitterness Units, used to gauge a beer's bitterness ("+50" will return everything above 50 IBUs, "-50" will return everything less than 50 IBUs, "30,50" will return everything between 30 and 50 IBUs inclusive)

Style

A beer style is a label given to a beer that describes its overall character and often times its origin
